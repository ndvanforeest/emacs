* Installing Emacs

1. Close Emacs.
2. =cd ~/.emacs.d=
2. =rm -rf elpa elpy=
3. =git clone  https://github.com/ndvanforeest/emacs.git=
4. Open Emacs, it will download all the packages. (Ignore the warnings on the first launch.)
5. And now you can use it.
